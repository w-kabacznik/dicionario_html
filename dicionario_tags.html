Dicionário de Tags HTML

<html>

Define o início e o fim de um documento HTML.

<!DOCTYPE html>
<html lang="en">
<head>
</head>
<body>
</body>
</html>

<h1> a <h6>

Define títulos de diferentes níveis de importância.

Definição e uso As tags " H1- H6 "são usadas para definir títulos HTML.

define o título mais importante -- Exemplo Tamanho H1.

define o título menos importante -- Exemplo tamanho H6.
Nota: Use apenas um -- h1 -- por página - isso deve representar o título/assunto principal para a página inteira. Além disso, não pule os níveis de título - comece com h1, depois use h2, e assim por diante.
<h1>Título 1</h1>
<h2>Título 2</h2>
<h3>Título 3</h3>
<h4>Título 4</h4>
<h5>Título 5</h5>
<h6>Título 6</h6>

<p>

Define um parágrafo de texto.

<p>Este é um parágrafo.</p>

<a>

Define um hiperlink.

<a href="https://www.exemplo.com">Clique aqui</a>

<img>

Insere uma imagem.

<img src="imagem.jpg" alt="Descrição da imagem">

<ul> e <ol> e <li>

<ul> define uma lista não ordenada, <ol> define uma lista ordenada, e <li> define um item de lista.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>

<section>

Define uma seção em um documento.

<section>
  <h2>Título da Seção</h2>
  <p>Conteúdo da seção.</p>
</section>

<article>

Define um conteúdo independente.

<article>
  <h2>Título do Artigo</h2>
  <p>Conteúdo do artigo.</p>
</article>

<aside>

Define um conteúdo tangencial ou secundário.

<aside>
  <p>Este é um conteúdo lateral.</p>
</aside>

<figure> e <figcaption>

<figure> define um conteúdo ilustrativo e <figcaption> adiciona uma legenda.

<figure>
  <img src="imagem.jpg" alt="Descrição da imagem">
  <figcaption>Legenda da imagem</figcaption>
</figure>

<main>

Define o conteúdo principal do documento.

<main>
  <h1>Título Principal</h1>
  <p>Conteúdo principal da página.</p>
</main>

<time>

Define uma data/hora.

<time datetime="2023-07-20">20 de julho de 2023</time>

<blockquote>

Define uma seção de citação.

<blockquote>
  <p>Este é um bloco de citação.</p>
  <footer>— Fonte da Citação</footer>
</blockquote>
